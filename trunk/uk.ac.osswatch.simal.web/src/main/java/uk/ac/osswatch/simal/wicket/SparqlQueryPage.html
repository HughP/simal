Example Queries

Query Description 
PREFIX  doap: <http://usefulinc.com/ns/doap#>
SELECT DISTINCT ?name
WHERE {
  ?project a doap:Project .
  ?project doap:name ?name
}
Get the name of all doap:Projects in the repository.
